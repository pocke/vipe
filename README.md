vipe
=======

vipe is "Vim pipe".

Can use Vim between pipe.


Installation
----------

```sh
go get github.com/pocke/vipe
```

And Vim is required.


Usage
------


```sh
$ cat hoge | vipe | sh
$ cat hoge | vipe -g | sh   # If use GVim
```
